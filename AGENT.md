This is a personal portfolio management app.

The app is split into a backend app and several, alternative frontend applications.

In the ./backend folder is the Python FastAPI backend.
In the ./alpinefrontend folder is the AlpineJS frontend.
In the ./textualfrontend folder is the Python Textual frontend.
in the ./vuefrontend folder is the VueJS frontend.

Currently only the backend and the Textual frontend are under active development.
